# Ghostie 安装指南

本指南将帮助您在各种操作系统上安装 Ghostie。

## Windows 安装

1. 下载 `.exe` 安装包
2. 双击安装包运行
3. 按照安装向导的提示进行操作
4. 安装完成后，可以从开始菜单或桌面快捷方式启动 Ghostie

```bash
# 也可以通过 Winget 安装
winget install Ghostie
```

## macOS 安装

1. 下载 `.dmg` 文件
2. 打开 `.dmg` 文件
3. 将 Ghostie 图标拖到 Applications 文件夹中
4. 从启动台或 Applications 文件夹启动 Ghostie

```bash
# 也可以通过 Homebrew 安装
brew install ghostie
```

## Linux 安装

### Ubuntu/Debian

```bash
# 添加 Ghostie 仓库
curl -fsSL https://ghostie.io/gpg.key | sudo gpg --dearmor -o /usr/share/keyrings/ghostie-archive-keyring.gpg
echo "deb [signed-by=/usr/share/keyrings/ghostie-archive-keyring.gpg] https://repo.ghostie.io/apt stable main" | sudo tee /etc/apt/sources.list.d/ghostie.list

# 更新软件包列表并安装
sudo apt update
sudo apt install ghostie
```

### Arch Linux

```bash
yay -S ghostie
```

### 其他 Linux 发行版

1. 下载 `.AppImage` 文件
2. 添加执行权限: `chmod +x Ghostie-x.y.z.AppImage`
3. 运行应用程序: `./Ghostie-x.y.z.AppImage`

## 疑难解答

如果在安装过程中遇到问题，请：

1. 确保您的系统满足[最低要求](/docs/start#系统要求)
2. 检查是否有足够的磁盘空间
3. 确保您有管理员/root 权限
4. 查看我们的[常见问题解答](/docs/faq)
5. 在[社区论坛](/docs/community)寻求帮助
