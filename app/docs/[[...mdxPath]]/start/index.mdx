# 开始使用 Ghostie

欢迎使用 Ghostie！本指南将帮助您快速上手，体验 Ghostie 的强大功能。

## 系统要求

Ghostie 支持以下操作系统：

- **Windows**: Windows 10 64位 或更高版本
- **macOS**: macOS 10.15 (Catalina) 或更高版本
- **Linux**: Ubuntu 20.04 LTS, Debian 10 或更高版本

## 下载安装

1. 前往 [下载页面](/docs/download) 获取适合您系统的安装包
2. 按照 [安装指南](/docs/start/installation) 完成安装
3. 参考 [初始配置](/docs/start/configuration) 进行个性化设置

## 下一步

安装完成后，建议您：

- 查看 [基础功能教程](/docs/guide) 了解软件的基本操作
- 加入 [社区](/docs/community) 获取支持和分享经验
- 探索 [高级功能](/docs/guide/advanced) 发掘更多可能性
