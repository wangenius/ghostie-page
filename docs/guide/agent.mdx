# Agent

## Overview

Agent is the core concept in Ghostie, representing an independent AI assistant instance with:

- Independent model configuration
- Dedicated knowledge base
- Customizable workflows
- Unique role settings


## Agent Configuration


1. Knowledge Base

   - Knowledge selection
   - Access permissions
   - Update strategy
   - Retrieval settings

2. Workflow configuration

   - Workflow selection
   - Trigger conditions
   - Execution rules
   - Error handling

3. Plugin configuration

   - Plugin selection
   - Plugin settings
   - Permission management
   - Dependency management
