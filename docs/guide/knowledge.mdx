# 知识库指南

## 知识库概述

知识库是 Ghostie 中存储和管理知识的核心组件。它允许你：

- 导入和管理各种文档
- 建立知识之间的关联
- 实现智能检索
- 支持实时更新

## 文档管理

### 支持的文档类型

- Markdown 文件 (.md)
- PDF 文档 (.pdf)
- Word 文档 (.doc, .docx)
- 文本文件 (.txt)
- HTML 文件 (.html)
- 其他格式（通过插件支持）

### 文档导入

1. 单个文件导入

   - 拖拽文件到界面
   - 使用文件选择器
   - 从剪贴板粘贴

2. 批量导入

   - 选择多个文件
   - 导入整个文件夹
   - 从 URL 导入

3. 导入选项
   - 文档分类
   - 标签管理
   - 元数据设置
   - 访问权限
