# Configuration

After the first launch of Ghostie, you can make some basic settings to optimize your experience.

## Initial Setup

When you first launch Ghostie, you will see a welcome interface that guides you through the following initial settings:

1. **Select language** - Ghostie supports multiple languages, including Chinese, English, etc.
2. **Select theme** - Bright theme or dark theme, or follow the system
3. **Set workspace** - Configure the default workspace location
4. **Performance settings** - Adjust resource usage based on your computer's performance
