# Model

## Overview

Model is the core of AI capabilities in Ghostie, it is responsible for:

- Understanding user input
- Generating response content
- Executing specific tasks
- Providing professional services

## Model Types

- Text Model

- Txt2Img Model
- Img2Img Model
- Audio Model
- Video Model
- Embedding Model
