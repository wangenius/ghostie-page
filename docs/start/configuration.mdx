# Ghostie 初始配置

首次启动 Ghostie 后，您可以进行一些基本设置以优化您的使用体验。

## 首次启动设置

首次启动 Ghostie 时，会出现欢迎界面，引导您完成以下初始设置：

1. **选择界面语言** - Ghostie 支持多种语言，包括中文、英文等
2. **选择主题** - 明亮主题或暗黑主题，也可以选择跟随系统
3. **设置工作区** - 配置默认的工作区位置
4. **性能设置** - 根据您的计算机性能调整资源使用

## 界面设置

您可以随时通过"设置"界面调整以下参数：

### 外观设置

```yaml
界面主题: 明亮 / 暗黑 / 跟随系统
字体大小: 小 / 中 / 大
界面密度: 紧凑 / 标准 / 宽松
```

### 性能设置

```yaml
启动时自动运行: 开启 / 关闭
后台运行模式: 开启 / 关闭
资源使用限制: 低 / 中 / 高
```

## 快捷键设置

Ghostie 提供了丰富的默认快捷键，您也可以自定义这些快捷键：

| 功能 | 默认快捷键 | 说明 |
| --- | --- | --- |
| 新建项目 | Ctrl+N | 创建新项目 |
| 保存 | Ctrl+S | 保存当前项目 |
| 搜索 | Ctrl+F | 搜索内容 |
| 设置 | Ctrl+, | 打开设置界面 |
| 帮助 | F1 | 显示帮助信息 |

## 高级配置

高级用户可以通过编辑配置文件进行更精细的设置：

```bash
# Windows 配置文件位置
%APPDATA%\Ghostie\config.yaml

# macOS 配置文件位置
~/Library/Application Support/Ghostie/config.yaml

# Linux 配置文件位置
~/.config/ghostie/config.yaml
```
